# こんだてLoop 第2フェーズリリース要件（2026-02-06 更新）

> MVP後に追加予定の機能。レシピカタログ・マイページを解禁する。

## 全体方針
- MVPの**献立表 / レシピ帳**に加え、**レシピカタログ / マイページ**を追加。
- **4画面構成**: 献立表 / レシピ帳 / レシピカタログ / マイページ
- モーダルの**発火タイミングはプロト準拠**。

## 献立表（ホーム）
- ヘッダーに**通知**と**冷蔵庫**を表示。
- 献立セットを登録すると**買い物リスト自動生成**。
  - 生成基準：**セットに必要な食材**かつ**冷蔵庫に無いもの**。
- 買い物リストで「買った」→ **冷蔵庫へ反映**。
- 買い物リスト/冷蔵庫は**手動追加**が可能。
- 冷蔵庫からの**削除**が可能、**削除履歴**は冷蔵庫内で閲覧できる。
- **次の献立**を設定できる。

## レシピ帳（プライベート）
- **レシピ / レシピセット**の2タブで閲覧。
- **フリータグを基準にカテゴリ分け**（カテゴリごとのページ）。
- 同一レシピが**複数カテゴリに出る**想定。
- **カテゴリはユーザーが追加/編集可能**。

## レシピカタログ（パブリック）
- 仕様はレシピ帳と共通（**レシピ/セット**を表示）。
- **既定カテゴリは削除不可**、ユーザー追加は可能。
- **有料レシピ/有料セットの販売**を初期から実施。
- **メンバーシップ機能は未リリース**（表示のみ）。入っていても加入不可。
- 並び替え/絞り込みは初期から実装。
- 詳細から**料理家ページへ遷移**できる。
- **購入/保存**は初期から利用可能。

## マイページ（初期は機能制限）
- アカウント種別は**ユーザー / クリエイター**の2種のみ。
- **プロフィール編集はクリエイターのみ**（表示名/自己紹介/公開ステータス）。
- **ダッシュボード**は初期から表示。
- **メンバーシップ表示は可能**だが**機能は未リリース**。
- **診断/アーカイブは準備中**表示。
- **購入履歴は表示**。

### マイページ（設定）
- **支払い登録**（クレカ登録）を初期から実装。
- **プッシュ通知ON/OFF**は初期から実装。
- **フォロー通知/アカウントタイプ/ヘルプ・フィードバック**は準備中。
